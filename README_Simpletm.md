# SimpleTM 系统用户指南

> **功能概要**
> 本系统是专为游戏汉化协作打造的文本与进度管理平台。系统支持项目凭证隔离，通过术语表、人称表和全自动进度看板，实现多人在线协同翻译与任务指派。同时，系统深度整合 VS Code 插件，支持配置云端同步与剧本资产的批量导入导出。

## 1. 首页 (Home)

### 1.1 用户信息
- 显示当前登录账号的用户名、昵称与系统权限。

### 1.2 项目列表
- **列表查看**：默认展示当前用户参与的项目。开启“显示所有项目”开关可查看系统内的全部项目。
- **项目搜索**：支持输入项目 ID 或描述进行列表筛选。
- **进入项目**：点击项目名称跳转至该项目的具体管理页面。
- **删除项目（仅限管理员）**：点击项目右侧的“删除”按钮并确认，可将该项目及其所有数据从系统中彻底删除。

### 1.3 创建项目（仅限管理员）
- 输入“项目ID”与“项目描述”，点击“创建”按钮即可新建项目。项目 ID 仅支持字母、数字、下划线及连字符。

### 1.4 加入项目
- 输入目标项目的“项目ID”与“项目密码”，点击“加入”按钮获取该项目的访问权限。

## 2. 项目 (Project) 页面

项目页面是单个汉化工程的管理核心，通过多个标签页（Tabs）集中管理翻译数据、人员权限以及项目配置。

### 2.1 基础信息与全局操作
- **信息查看**：显示当前项目名称以及当前登录用户在该项目中的权限级别。
- **删除项目（仅限管理员）**：提供二次确认的“删除项目”功能，需输入完整的项目名称进行验证
。
### 2.2 标签页功能概括
- **术语/人称**：统一管理、维护游戏中的专属术语表以及角色人称表。
- **组员**：查看并管理该项目的参与人员，进行成员权限的分配与调整。
- **进度**：查看整个项目的翻译、校对进度以及全局字数统计。
- **设置**：配置项目的底层核心参数（如项目密码、文本解析规则等）。

### 2.3 术语表 (Translation Tab)
- **术语查看与排序**：展示当前项目的所有术语条目。支持按“添加顺序”或“原文拼音/字母”进行列表排序。
- **单条添加术语**：在列表底部手动输入“原文”、“译文”与“备注”，点击添加按钮录入新术语。
- **批量导入术语**：在“添加原文”输入框中，支持直接从 Excel 或表格软件中复制多行数据并粘贴，系统会自动识别并批量解析为多条术语。
- **术语编辑与修改**：直接点击任意术语的译文或备注文本，即可原地进行实时修改。
- **删除术语**：点击术语右侧的删除图标，可将该条术语从项目中移除。

### 2.4 人称表 (Naming Tab)
- **人称查看与分类**：分为“默认称呼”与“特殊称呼（指定称呼人）”两个列表，用于管理角色之间的称谓关系。
- **添加默认称呼**：输入“被称呼人”、“被称呼人翻译”与“备注”，用于设定该角色通用的标准译名。
- **添加特殊称呼**：输入“称呼人”、“被称呼人”、“被称呼人翻译”与“备注”，用于设定特定角色在称呼他人时的专属特殊译名（如特定角色间的特殊昵称）。
- **内容修改与删除**：可直接点击任意格子的文本进行原地修改，点击右侧删除图标可移除该名称映射规则。

### 2.5 成员 (Users Tab)
- **成员列表查看**：展示当前项目的所有参与人员，包括用户名、系统分配的权限级别以及在本项目中的昵称。
- **添加真实用户**：输入已注册的“用户名”，并为其分配项目权限（只读，编辑，管理员），点击按钮将其加入项目。
- **创建虚拟用户**：支持直接输入名称生成“虚拟用户”，用于快速添加还没注册的用户。添加的虚拟用户只在当前项目有效。
- **修改成员权限**：通过成员右侧的下拉菜单，可实时调整其在本项目中的权限级别。
- **修改项目昵称**：支持直接修改成员在本项目内的专属昵称。如果填写的昵称与已有的虚拟用户重名，系统会提示并支持将该虚拟用户合并注销。
- **生成共享链接**：点击按钮可生成一个专属于当前项目的快捷加入链接。复制该链接发送给其他组员，对方在vscode的dltxt插件中填写后即可直接加入本项目。
- **移除成员**：点击成员右侧的“删除”按钮并确认，可将其从当前项目中移除。

## 3. 进度 (Progress)

进度面板用于拆分文本任务、追踪翻译/校对各阶段完成度，并管理对应的剧本资产。

### 3.1 核心数据看板
- **全局进度条**：界面顶部实时显示整个项目的总翻译进度百分比。
- **任务状态追踪**：列表中的每一行代表一个独立的剧本或文本任务。用户可直接在格子里输入当前的具体进度数值，或勾选完成。
- **备注管理**：支持为每个独立任务填写文本备注，点击文字即可原地编辑修改。

### 3.2 任务与流程管理
- **添加/删除任务**：支持单条新建任务（输入任务名称）；点击行尾的删除图标可移除该任务。
- **管理进度阶段（列）**：支持在看板中添加新的工序阶段（如初翻、校对、润色），或者删除不需要的阶段。
- **清空看板**：支持一键清空当前看板的所有任务。执行时需要输入特定提示词进行二次确认。

### 3.3 人员指派 (Assign)
- **单项指派**：可直接点击某个进度格子内的头像图标，将该工序指派给特定的组员。
- **批量指派**：支持在列表左侧勾选多个任务，一键批量分发给同一位翻译或校对人员。

### 3.4 附件管理 (Attachment)
- **上传附件**：点击单元格内的上传图标，或将文本直接拖拽进单元格，可将对应的文本或剧本文件上传到该任务中。如果上传的文件名与任务名称不一致，系统会弹出确认弹窗提示，防止传错。
- **打包下载**：支持勾选多条任务，一键批量下载对应的剧本附件。

### 3.5 表格导入与导出 (Excel)
- **Excel 批量导入**：支持上传现成的 `.xlsx` 表格，系统会自动解析并批量生成对应的网页任务看板与进度阶段。
- **看板数据导出**：支持将当前的整个进度看板、人员指派和字数统计一键导出为标准的 Excel 表格。

## 4. 设置 (Config)
- **修改项目密码**：支持实时查看与修改当前项目的加入密码（项目凭证）。
- **覆盖导入新文本**：支持通过文件上传控件选择并导入新的汉化剧本文本（此操作会覆盖原有文本数据）。
- **VSCode 配置备份与恢复**：支持上传 `.json` 格式的 VSCode 插件配置文件，用于在服务器端统一组内的文本高亮与解析规则，或一键将当前配置导出下载为本地 JSON 文件。
- **进度表功能开关**：提供一键切换按钮，支持开启或关闭进度表的进阶管理功能（如任务看板、文件指派与附件管理等）。
